# Welcome

Welcome to **CloudSync Desktop**, a secure cloud storage application that enables you to store, organize, synchronize, and share files across multiple devices.

This guide explains how to use CloudSync Desktop's core features and helps you get started quickly.

---

# System Requirements

CloudSync Desktop supports the following operating systems:

| Platform | Minimum Version |
|----------|-----------------|
| Windows | Windows 10 |
| macOS | macOS 12 Monterey |
| Linux | Ubuntu 22.04 LTS |

Minimum hardware requirements:

- 4 GB RAM
- 500 MB available disk space
- Stable internet connection

---

# Getting Started

After installing CloudSync Desktop:

1. Launch the application.
2. Sign in using your CloudSync account.
3. Verify your email address.
4. Choose a synchronization folder.
5. Complete the setup wizard.

You are now ready to begin using CloudSync.

# Uploading Files

To upload files:

1. Open **Files**.
2. Click **Upload**.
3. Select one or more files.
4. Click **Open**.
5. Wait for synchronization to complete.

Supported file types include:

- PDF
- DOCX
- XLSX
- PPTX
- PNG
- JPG
- ZIP

---

# Creating Folders

To create a new folder:

1. Open **Files**.
2. Select **New Folder**.
3. Enter a folder name.
4. Click **Create**.

Folders can be renamed, moved, or deleted at any time.

---

# Sharing Files

CloudSync allows secure file sharing.

To share a file:

1. Right-click the file.
2. Select **Share**.
3. Enter the recipient's email.
4. Assign permissions:
   - Viewer
   - Editor
5. Click **Send Invitation**.

---

# Managing Notifications

Notification preferences can be customized.

Navigate to:

**Settings → Notifications**

You can enable or disable notifications for:

- File uploads
- Shared files
- Login alerts
- Synchronization status
- Product updates

# Frequently Asked Questions

## Why are my files not syncing?

Check your internet connection and verify that synchronization is enabled.

---

## Can I recover deleted files?

Yes.

Deleted files remain in the Recycle Bin for 30 days.

---

## How much storage is available?

Storage depends on your subscription plan.

View available storage under:

**Settings → Storage**

---

## Can I use CloudSync offline?

Yes.

Changes made offline automatically synchronize when your device reconnects to the internet.
