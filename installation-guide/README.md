

# Windows Installation

Follow these steps to install CloudSync Desktop on Windows.

## Step 1: Download the Installer

Download **CloudSyncSetup.exe** from the official download page.

> **Note:** Ensure you download the latest stable release.

---

## Step 2: Launch the Installer

Double-click **CloudSyncSetup.exe**.

If prompted by Windows User Account Control (UAC), click **Yes**.

---

## Step 3: Accept the License Agreement

Read the End User License Agreement (EULA).

Select:

☑ I accept the License Agreement

Click **Next**.

---

## Step 4: Choose Installation Location

The default installation directory is:

```text
C:\Program Files\CloudSync\
```

Click **Next**.

---

## Step 5: Install

Click **Install**.

The installation process usually takes less than two minutes.

---

## Step 6: Finish

Click **Finish**.

Launch CloudSync Desktop from the Start Menu.

---

# macOS Installation

## Step 1

Download the **CloudSync.dmg** installer.

---

## Step 2

Open the downloaded DMG file.

---

## Step 3

Drag the CloudSync application into the **Applications** folder.

---

## Step 4

Open Launchpad.

Click **CloudSync Desktop**.

If macOS displays a security warning, select:

**Open Anyway**

under:

System Settings → Privacy & Security

---

## Step 5

Sign in using your company credentials.

---

# Linux Installation (Ubuntu)

Install the application using the terminal.

```bash
sudo dpkg -i cloudsync.deb
```

If dependency errors occur, run:

```bash
sudo apt --fix-broken install
```

Launch CloudSync using:

```bash
cloudsync
```

---

# First-Time Configuration

After launching CloudSync Desktop:

1. Sign in.
2. Verify your email.
3. Select a synchronization folder.
4. Choose synchronization preferences.
5. Complete the setup wizard.

CloudSync will begin synchronizing your files automatically.
